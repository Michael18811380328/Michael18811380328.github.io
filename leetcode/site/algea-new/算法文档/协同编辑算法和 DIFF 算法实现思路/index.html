    еҚҸеҗҢзј–иҫ‘з®—жі•е’Ң DIFF з®—жі•е®һзҺ°жҖқи·Ҝ¶

    еҜ№еҚҸеҗҢзј–иҫ‘е’Ң diff зҡ„дёҖзӮ№жғіжі•и®°еҪ•

    sockct еҗҢжӯҘ¶

    socket дё»иҰҒеә”з”ЁеңЁеӨҡдәәиҒҠеӨ©е®ӨпјҢеңЁзәҝеҠЁжҖҒиҜ„и®әпјҢж–Үд»¶еҚҸеҗҢзј–иҫ‘зӯүеңәеҗҲпјҢдҪҝз”Ё socket еҚҸи®®гҖӮ

    жңҚеҠЎз«Ҝе’ҢеӨҡдёӘе®ўжҲ·з«ҜеҰӮдҪ•еҗҢжӯҘпјҹзұ»дјјзҪ‘з»ңдёӯе№ҝж’ӯжҠҘж–Үзҡ„жҖқи·Ҝпјҡе®ўжҲ·з«ҜеҸ‘йҖҒдёҖдёӘж¶ҲжҒҜпјҢжңҚеҠЎз«Ҝж”¶еҲ°ж¶ҲжҒҜеҗҺпјҢж”ҫе…Ҙж¶ҲжҒҜе Ҷж ҲдёӯпјҢеҗҢж—¶е№ҝж’ӯз»ҷе…ЁйғЁе®ўжҲ·з«ҜгҖӮе…¶д»–е®ўжҲ·з«Ҝж”¶еҲ°е№ҝж’ӯж¶ҲжҒҜеҗҺпјҢйҖҡиҝҮиҜ„и®әж—¶й—ҙпјҢжӣҙж–°жң¬ең°зҡ„иҜ„и®әеҲ—иЎЁгҖӮ

    еӣ дёәдёҚеҗҢе®ўжҲ·з«Ҝзҡ„иҜ„и®әдёҚдјҡдә’зӣёж„ҹжҹ“пјҢжүҖд»ҘиҝҷйҮҢдёҚж¶үеҸҠ diffпјҢзӣҙжҺҘжҠҠж–°иҺ·еҸ–зҡ„иҜ„и®әпјҢpush еҲ°е·ІжңүиҜ„и®әжңҖеҗҺеҚіеҸҜпјҲж—¶й—ҙеӨҚжқӮеәҰO1пјүгҖӮеҰӮжһңеҗҢж—¶ж”¶еҲ°еӨҡжқЎиҜ„и®әпјҢжҲ–иҖ…з”ұдәҺзҪ‘з»ңеҺҹеӣ пјҢжңҚеҠЎз«ҜеҸ‘йҖҒиҜ„и®әзҡ„йЎәеәҸе’Ңе®ўжҲ·з«ҜжҺҘж”¶зҡ„иҜ„и®әйЎәеәҸдёҚдёҖж ·пјҢйӮЈд№Ҳжң¬ең°йңҖиҰҒжҹҘжүҫеҲ°еҗҲйҖӮзҡ„дҪҚзҪ®е№¶жҸ’е…ҘиҜ„и®әпјҲж—¶й—ҙеӨҚжқӮеәҰ OLogNпјү

    еҚҸеҗҢзј–иҫ‘ diff з®—жі•¶

    еҚҸеҗҢпјҡеӨҡдёӘе®ўжҲ·з«ҜеҜ№еҗҢдёҖдёӘж•°жҚ®жӣҙж”№пјҢеҸ‘йҖҒеҲ°жңҚеҠЎз«ҜпјҢжҲ–иҖ…еңЁе®ўжҲ·з«Ҝд№Ӣй—ҙеҗҢжӯҘпјҢеҰӮдҪ•е®һзҺ°еҗҲе№¶ diff ж“ҚдҪңпјҹ

    1гҖҒж•°жҚ®зұ»еһӢжҳҜз®ҖеҚ•ж јејҸпјҲж•°еӯ—пјҢеӯ—з¬ҰдёІпјүпјҡж №жҚ®ж—¶й—ҙйЎәеәҸпјҢзӣҙжҺҘиҰҶзӣ–

    дҫӢеҰӮ 1 -> 2, 1-> 3 иҝҷдёӨдёӘиЎҢдёәпјҢж №жҚ®ж—¶й—ҙйЎәеәҸпјҢdiff еҗҺзӣҙжҺҘеҸҳжҲҗ 3пјҢиҰҶзӣ–еҺҹжқҘзҡ„ 2

    2гҖҒж•°жҚ®зұ»еһӢжҳҜж•°з»„пјҡйҒҚеҺҶж•°з»„пјҢеҰӮжһңдёҚеҗҢеҶ…е®№пјҢеҲӨж–ӯеўһеҲ жғ…еҶөпјҢе®һдҫӢжҳҜ git ж“ҚдҪң

    дҫӢеҰӮпјҢдёҖдёӘж–Үд»¶жҳҜ

    const a = 10;
const b = 20;
const c = 30;
const d = 40;

    еҸҰдёҖдёӘж–Үд»¶

    const a = 10;
const b = 10;
const c = 30;
const e = 50;

    йӮЈд№Ҳ e е°ұжҳҜж•°з»„ж–°еўһзҡ„дёҖйЎ№пјҢb еҶ…е®№еҸ‘з”ҹеҸҳеҢ–пјҢйңҖиҰҒжүӢеҠЁжҜ”иҫғпјҢеҲӨж–ӯдҝқз•ҷ master иҝҳжҳҜдҪҝз”Ёж–°зҡ„ code

    * const a = 10;

    3 ж•°жҚ®зұ»еһӢжҳҜеӨҚжқӮеҜ№иұЎпјҢдҫӢеҰӮ DOM ж ‘жҲ–иҖ… React дёӯиҷҡжӢҹ DOM

    йҰ–е…ҲжҜ”иҫғж ‘иҠӮзӮ№зҡ„зұ»еһӢпјҢеҰӮжһңзұ»еһӢдёҚдёҖиҮҙпјҢзӣҙжҺҘеҲ йҷӨж—§иҠӮзӮ№пјҢдҪҝз”Ёж–°иҠӮзӮ№жёІжҹ“пјӣ

    еҰӮжһңиҠӮзӮ№зұ»еһӢдёҖиҮҙпјҲйғҪжҳҜ divпјүйӮЈд№Ҳз»§з»ӯжҜ”иҫғиҠӮзӮ№зҡ„дёӢеұӮиҠӮзӮ№ div.childrenпјҢдёҖзӣҙеҲ°жҜ”иҫғе®Ңж•ҙж ‘иҠӮзӮ№пјӣ
