    Fragments¶

    React дёӯзҡ„дёҖдёӘеёёи§ҒжЁЎејҸжҳҜдёҖдёӘз»„д»¶иҝ”еӣһеӨҡдёӘе…ғзҙ гҖӮFragments е…Ғи®ёдҪ е°ҶеӯҗеҲ—иЎЁеҲҶз»„пјҢиҖҢж— йңҖеҗ‘ DOM ж·»еҠ йўқеӨ–иҠӮзӮ№гҖӮ

    render() {
  return (
    <React.Fragment>
      <ChildA />
      <ChildB />
      <ChildC />
    </React.Fragment>
  );
}

    еҠЁжңә¶

    дёҖз§Қеёёи§ҒжЁЎејҸжҳҜз»„д»¶иҝ”еӣһдёҖдёӘеӯҗе…ғзҙ еҲ—иЎЁгҖӮд»ҘжӯӨ React д»Јз ҒзүҮж®өдёәдҫӢпјҡ

    class Table extends React.Component {
  render() {
    return (
      <table>
        <tr>
          <Columns />
        </tr>
      </table>
    );
  }
}

    <Columns /> йңҖиҰҒиҝ”еӣһеӨҡдёӘ <td> е…ғзҙ д»ҘдҪҝжёІжҹ“зҡ„ HTML жңүж•ҲгҖӮеҰӮжһңеңЁ <Columns /> зҡ„ render() дёӯдҪҝз”ЁдәҶзҲ¶ divпјҢеҲҷз”ҹжҲҗзҡ„ HTML е°Ҷж— ж•ҲгҖӮ

    class Columns extends React.Component {
  render() {
    return (
      <div>
        <td>Hello</td>
        <td>World</td>
      </div>
    );
  }
}

    еҫ—еҲ°дёҖдёӘ <Table /> иҫ“еҮәпјҡ

    <table>
  <tr>
    <div>
      <td>Hello</td>
      <td>World</td>
    </div>
  </tr>
</table>

    Fragments и§ЈеҶідәҶиҝҷдёӘй—®йўҳгҖӮ

    зҹӯиҜӯжі•¶

    дҪ еҸҜд»ҘдҪҝз”ЁдёҖз§Қж–°зҡ„пјҢдё”жӣҙз®Җзҹӯзҡ„иҜӯжі•жқҘеЈ°жҳҺ FragmentsгҖӮе®ғзңӢиө·жқҘеғҸз©әж Үзӯҫпјҡ

    class Columns extends React.Component {
  render() {
    return (
      <>        
        <td>Hello</td>
        <td>World</td>
      </>    
    );
  }

    дҪ еҸҜд»ҘеғҸдҪҝз”Ёд»»дҪ•е…¶д»–е…ғзҙ дёҖж ·дҪҝз”Ё <>пјҢйҷӨдәҶе®ғдёҚж”ҜжҢҒ key жҲ–еұһжҖ§гҖӮ

    еёҰ key зҡ„ Fragments¶

    дҪҝз”ЁжҳҫејҸ <React.Fragment> иҜӯжі•еЈ°жҳҺзҡ„зүҮж®өеҸҜиғҪе…·жңү keyгҖӮдёҖдёӘдҪҝз”ЁеңәжҷҜжҳҜе°ҶдёҖдёӘйӣҶеҗҲжҳ е°„еҲ°дёҖдёӘ Fragments ж•°з»„

    дёҫдёӘдҫӢеӯҗпјҢеҲӣе»әдёҖдёӘжҸҸиҝ°еҲ—иЎЁпјҡ

    function Glossary(props) {
  return (
    <dl>
      {props.items.map(item => (
        // жІЎжңү`key`пјҢReact дјҡеҸ‘еҮәдёҖдёӘе…ій”®иӯҰе‘Ҡ
        <React.Fragment key={item.id}>
          <dt>{item.term}</dt>
          <dd>{item.description}</dd>
        </React.Fragment>
      ))}
    </dl>
  );
}

    key жҳҜе”ҜдёҖеҸҜд»Ҙдј йҖ’з»ҷ Fragment зҡ„еұһжҖ§гҖӮжңӘжқҘжҲ‘д»¬еҸҜиғҪдјҡж·»еҠ еҜ№е…¶д»–еұһжҖ§зҡ„ж”ҜжҢҒпјҢдҫӢеҰӮдәӢд»¶гҖӮ
