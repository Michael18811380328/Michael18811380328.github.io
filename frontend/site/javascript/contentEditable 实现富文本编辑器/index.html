    зҺ°еңЁдё»жөҒзҡ„webеҜҢж–Үжң¬зј–иҫ‘еҷЁеҮ д№ҺйғҪжҳҜеҹәдәҺcontentEditableзҡ„пјҢжҲ‘д»¬д№ӢеүҚзҡ„зј–иҫ‘еҷЁеј•ж“Һе°ұжҳҜеҹәдәҺTinyMCEйӯ”ж”№зҡ„пјҢиҝҷдёӘеј•ж“Һзӣ®еүҚеҸӘжңүжҲ‘дёҖдёӘдәәиҙҹиҙЈгҖӮеҜҢж–Үжң¬зј–иҫ‘еҷЁжҳҜжҲ‘д»¬зҡ„ж ёеҝғдёҡеҠЎпјҢзҺ°еңЁleaderиҰҒжҗһдёҖдёӘдё“й—Ёзҡ„еҜҢж–Үжң¬зј–иҫ‘еҷЁеӣўйҳҹпјҢејҖеҸ‘зұ»дјјдәҺGoogle docsйӮЈз§ҚдёҚеҹәдәҺcontentEditableзҡ„зј–иҫ‘еҷЁеј•ж“ҺпјҢиҝҷе®Ңе…ЁжҳҜеҸҰдёҖдёӘзә§еҲ«зҡ„дәӢжғ…дәҶпјҢжҲ‘жүҫйҒҚдәҶе…ЁзҪ‘д№ҹжүҫдёҚеҲ°зұ»дјјиҝҳжҙ»зқҖзҡ„ејҖжәҗйЎ№зӣ®гҖӮйҷӨдәҶд»Јз Ғзј–иҫ‘еҷЁaceд№ӢеӨ–зңҹзҡ„дёҖж— жүҖиҺ·пјҢжңүжІЎжңүеҒҡиҝҮзҡ„еӨ§дҪ¬зЁҚеҫ®и®ІдёҖдёӢжҖқи·ҜпјҢиҰҒе®һзҺ°йӮЈдәӣе…ій”®з»„д»¶пјҹжҲ–иҖ…и®ІдёҖдёӢеӨ§е®¶еӣўйҳҹдёҖејҖе§ӢжҳҜжҖҺд№Ҳе…ҘжүӢзҡ„пјҹ

    й“ҫжҺҘпјҡhttps://www.zhihu.com/question/366666295/answer/976815981

    и§ЈеҶі¶

    иғҪжҸҗеҮәиҝҷдёӘй—®йўҳпјҢиҜҙжҳҺйўҳдё»жңүиҝҮеҫҲд»”з»Ҷзҡ„жҖқиҖғгҖӮжҲ‘е°қиҜ•еҖҹжӯӨжҫ„жё…дёҖдёӘеӨ§е®¶е…Ҙй—Ё Web еҜҢж–Үжң¬зј–иҫ‘ж—¶еҸҜиғҪзҡ„иҜҜи§Јеҗ§пјҡ

    еңЁжөҸи§ҲеҷЁйҮҢпјҢжү“ејҖдәҶ contentEditable дёҚзӯүдәҺ**еҖҹеҠ©дәҶ** contentEditableгҖӮиҝҷжҳҜд»Җд№Ҳж„ҸжҖқе‘ўпјҹ

    • жү“ејҖдәҶ contentEditableпјҢж„Ҹе‘ізқҖйЎөйқўиҝҷдёӘеҢәеҹҹеҶ…**е…·еӨҮдәҶиҝӣиЎҢеҜҢж–Үжң¬зј–иҫ‘зҡ„иғҪеҠӣ**гҖӮ
    • еҖҹеҠ©дәҶ contentEditableпјҢж„Ҹе‘ізқҖ**дҫқиө–дәҶжөҸи§ҲеҷЁй»ҳи®Өзҡ„еҜҢж–Үжң¬зј–иҫ‘иЎҢдёә**гҖӮ

    зІ—з•ҘзңӢжқҘпјҢдёҖдёӘ DOM е…ғзҙ еҠ дёҠ contentEditable еұһжҖ§еҗҺпјҢе°ұеӨ§иҮҙе…·еӨҮдәҶиҝҷдәӣиғҪеҠӣпјҡ

    • йҖүеҢәзҡ„жӢ–йҖүзј–иҫ‘е’Ңе…үж Үй—ӘеҠЁж”ҜжҢҒ
    • ж–№еҗ‘й”®жҺ§еҲ¶ж—¶зҡ„й»ҳи®ӨиЎҢдёә
    • иҫ“е…Ҙж–Үеӯ—ж—¶зҡ„й»ҳи®ӨиЎҢдёәпјҲзүөжүҜеҲ°иҫ“е…Ҙжі•пјү
    • еӨҚеҲ¶зІҳиҙҙж—¶зҡ„й»ҳи®ӨиЎҢдёәпјҲзүөжүҜеҲ°еҜҢж–Үжң¬еүӘиҙҙпјү

    дё»жөҒжөҸи§ҲеҷЁйғҪдјҡз»ҷдҪ дёҖеҘ—ејҖз®ұеҚіз”Ёзҡ„ж–№жЎҲпјҢи®©дҪ з”ЁдёҖиЎҢд»Јз ҒеҚіеҸҜе®һзҺ°еҜҢж–Үжң¬зј–иҫ‘еҷЁгҖӮдҪҶеӨ©дёӢжІЎжңүе…Қиҙ№зҡ„еҚҲйӨҗпјҢеғҸиҝҷз§ҚжңҖз®ҖеҚ•зҡ„ж–№жЎҲе°ұжңүдј иҜҙдёӯзҡ„дёҚе…је®№зҡ„й—®йўҳгҖӮжҜ”еҰӮеҗҢж ·дёҖдёӘеҠ зІ—ж“ҚдҪңпјҢжІЎдәәи§„е®ҡжөҸи§ҲеҷЁжҳҜеә”иҜҘз»ҷж–Үжң¬еҘ—дёӘ <b> ж ҮзӯҫиҝҳжҳҜз»ҷ <span> еҠ дёӘ font-weight:bold; зҡ„ CSS ж ·ејҸжқҘе®һзҺ°пјҲеҚ°иұЎйҮҢ Chrome жҳҜзӣҙжҺҘеҘ— span зҡ„пјҢиҜҙеҘҪзҡ„иҜӯд№үеҢ–е‘ўпјүгҖӮдәҺжҳҜжңҖеҗҺеңЁ Chrome е’Ң Firefox йҮҢдә’зӣёзј–иҫ‘зҡ„дә§зү©пјҢе°ұеҝ…з„¶жҳҜдёҚеҸҜжҺ§гҖҒдёҚеҸҜйў„жңҹзҡ„дәҶгҖӮеҫҲеӨҡдәәиҜҙеҜҢж–Үжң¬еҒҡиө·жқҘеҫҲи„ҸпјҢиҝҷе°ұжҳҜеҺҹеӣ д№ӢдёҖгҖӮ

    жүҖд»ҘиҝҳжҳҜйӮЈеҸҘиҖҒиҜқпјҢе®Ңе…Ёдҫқиө–еҲ«дәәзҺ°жҲҗзҡ„жҠҖжңҜжҳҜдјҡиў«еҚЎи„–еӯҗзҡ„гҖӮзӣёеә”ең°пјҢFacebook зҡ„ DraftJS иҷҪз„¶дёҚеӨӘеҘҪз”ЁпјҢдҪҶд»ЈиЎЁдәҶдёҖз§Қ React жҖқжҪ®еј•йўҶдёӢзҡ„еҜҢж–Үжң¬ж–°жҖқи·ҜпјҲеӨ§жҰӮзҺҮдёҚжҳҜе®ғйҰ–еҲӣпјҢдҪҶиҝҷд№ҲиҜҙиө·жқҘж–№дҫҝеүҚз«ҜеҗҢеӯҰд»¬зҗҶи§ЈпјүпјҢз®ҖеҚ•жқҘиҜҙжҳҜиҝҷж ·зҡ„пјҡ

    1. еҗҢж ·з”Ё contentEditableпјҢдҪҶеҸӘе…іеҝғе®ғзҡ„йҖүеҢә (Selection) зҠ¶жҖҒгҖӮ
    2. иҫ“е…Ҙж–Үжң¬ж—¶пјҢд»Ҙ nextState = f(selection, input) зҗҶеҝөи®Ўз®—еҮәж–°зҠ¶жҖҒгҖӮ

    иҝҷж ·зңӢжқҘпјҢж•ҙдёӘзј–иҫ‘еҷЁе°ұеғҸжҳҜдёҖдёӘ React дёӯзҡ„еҸ—жҺ§з»„д»¶дәҶгҖӮе…ій”®зҡ„еҸ—жҺ§иЎҢдёәеӨ§жҰӮеҢ…жӢ¬иҝҷдәӣпјҡ

    • жҢүй”®иҫ“е…Ҙиў«жӢҰжҲӘпјҢеҹәдәҺ f(selection, input) и®Ўз®—еҮәж–°зҠ¶жҖҒ
    • еӨҚеҲ¶зІҳиҙҙиў«жӢҰжҲӘпјҢеҹәдәҺ f(selection, input) и®Ўз®—еҮәж–°зҠ¶жҖҒ
    • DOM жӣҙж”№иў«жӢҰжҲӘпјҢеҹәдәҺ nextState еҚ•еҗ‘ең°жёІжҹ“еҮә DOM зҠ¶жҖҒ

    йӮЈд№ҲпјҢиҝҳжңүе“Әдәӣең°ж–№йңҖиҰҒдҫқиө– contentEditable е‘ўпјҹе…¶е®һе°ұжҳҜе’Ң Selection ејәзӣёе…ізҡ„дёңиҘҝпјҡ

    • **йҖүеҢәй«ҳдә®зҠ¶жҖҒ**дҫқиө– contentEditableпјҢеҗҰеҲҷдҪ йңҖиҰҒиҮӘе·ұжёІжҹ“йӮЈдёӘгҖҢжӢ–и“қгҖҚеҢәеҹҹгҖӮ
    • **зӮ№еҮ»еҗҺзҡ„йҖүдёӯзҠ¶жҖҒ**дҫқиө– contentEditableпјҢеҗҰеҲҷдҪ йңҖиҰҒиҮӘе·ұи®Ўз®—жҹҗдёӘеқҗж ҮдёӢеҜ№еә”дәҶе“ӘдёӘж–Үеӯ—пјҢж„Ҹе‘ізқҖиҰҒиҮӘе·ұеҺ»и§Јжһҗеӯ—дҪ“еҸӮж•°еҒҡж–Үжң¬жҺ’зүҲпјҢиҝҷжҳҜеӨ©еқ‘гҖӮ
    • **ж–№еҗ‘й”®ж“ҚдҪңеҗҺзҡ„зҠ¶жҖҒ**дҫқиө– contentEditableпјҢзҗҶз”ұеҗҢдёҠгҖӮ

    жҲ‘зӣёдҝЎзңӢеҲ°иҝҷйҮҢпјҢжңүз»ҸйӘҢзҡ„еүҚз«ҜеҗҢеӯҰе·Із»ҸзҹҘйҒ“жҖҺж ·ејҖе§Ӣе°қиҜ•гҖҢдёҚеҖҹеҠ© contentEditable зҡ„й»ҳи®ӨиЎҢдёәжқҘе®һзҺ°еҜҢж–Үжң¬зј–иҫ‘еҷЁгҖҚдәҶеҗ§гҖӮеҪ“з„¶пјҢжҲ‘е·Із»ҸжҢәд№…жІЎеҒҡеҜҢж–Үжң¬дәҶпјҢдёҠйқўдёҖдәӣжҰӮеҝөеҸҜиғҪдёҺе…·дҪ“з»ҶиҠӮжңүдәӣеҮәе…ҘпјҢж¬ўиҝҺеӨ§е®¶жҢҮжӯЈгҖӮ

    жіЁж„ҸпјҢи®ёеӨҡе®һйҷ…е·ҘзЁӢеҒҡиө·жқҘпјҢз»қдёҚжҳҜзҹҘйҒ“дёӘеҺҹзҗҶе°ұеӨҹдәҶгҖӮеҜҢж–Үжң¬еңЁз»ҶиҠӮеұӮйқўжңүйқһеёёеӨҡйә»зғҰзҡ„ең°ж–№пјҢйҷӨйқһе®һеҠӣйӣ„еҺҡеҰӮ GoogleпјҢеҗҰеҲҷдёҚжҺЁиҚҗеӨ§е®¶еңЁе•ҶдёҡйЎ№зӣ®йҮҢе®Ңе…Ёд»ҺеӨҙеҒҡиө·гҖӮе…·дҪ“еҸҜеҸӮиҖғжҲ‘зҡ„иҝҷдёӘеӣһзӯ”пјҡ жңүеӨҡеӨ§жҜ”дҫӢзҡ„еүҚз«Ҝе·ҘзЁӢеёҲпјҢиғҪеңЁеҗҲзҗҶж—¶й—ҙеҶ…зӢ¬з«ӢејҖеҸ‘еҮәдёҖдёӘи¶ід»Ҙе•Ҷз”Ёзҡ„еҜҢж–Үжң¬зј–иҫ‘еҷЁпјҹ

    е…·дҪ“еҲ°е®һйҷ…йЎ№зӣ®пјҢProseMirror е’Ң Slate зҡ„жЎҶжһ¶еҢ–и®ҫи®ЎйғҪеҫҲдёҚй”ҷгҖӮжҲ‘зӣёдҝЎе®ғд»¬жҳҜејҖеҸ‘гҖҢиҮӘдё»еҸҜжҺ§гҖҚзҡ„еҜҢж–Үжң¬зј–иҫ‘еҷЁж—¶зҡ„жӯЈи§ЈгҖӮи®ёеӨҡеҺҶеҸІз»ҸйӘҢе·Із»Ҹе‘ҠиҜүжҲ‘д»¬пјҢиҮӘдё»еҸҜжҺ§жңӘеҝ…ж„Ҹе‘ізқҖе®Ңе…ЁиҮӘе·ұд»ҺеӨҙеҶҷпјҢиҖҢжҳҜиҰҒжҺҢжҺ§еҜ№ж»Ўи¶іиҮӘе·ұжңӘжқҘйңҖжұӮзҡ„жҺ§еҲ¶жқғгҖӮеңЁиҝҷе°Ҹе°Ҹзҡ„еҜҢж–Үжң¬зј–иҫ‘еҷЁйҮҢпјҢдёҚд№ҹжҳҜиҝҷж ·еҗ—пјҹ
