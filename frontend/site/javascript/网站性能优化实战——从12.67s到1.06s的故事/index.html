    пјҲPSпјҡдҪҝз”Ёchromeеә”з”Ёе•Ҷеә—йңҖиҰҒзҝ»еўҷпјҢжҖҺд№Ҳзҝ»еўҷжҲ‘е°ұдёҚдҫҝеӨҡиҜҙдәҶпјү

    иҝҷе°ұжҳҜPage Speedзҡ„еҠҹиғҪз•Ңйқўпјҡ

    жҲ‘д»¬еҸӘйңҖиҰҒжү“ејҖеҫ…жөӢиҜ•зҡ„зҪ‘йЎөпјҢз„¶еҗҺзӮ№еҮ»Page SpeedйҮҢзҡ„ Start analyzingжҢүй’®пјҢе®ғе°ұдјҡиҮӘеҠЁеё®жҲ‘д»¬жөӢиҜ•зҪ‘з»ңдј иҫ“жҖ§иғҪдәҶпјҢиҝҷжҳҜжҲ‘зҡ„зҪ‘з«ҷжөӢиҜ•з»“жһңпјҡ

    Page SpeedжңҖдәәжҖ§еҢ–зҡ„ең°ж–№пјҢдҫҝжҳҜе®ғдјҡеҜ№жөӢиҜ•зҪ‘з«ҷзҡ„жҖ§иғҪз“¶йўҲжҸҗеҮәе®Ңж•ҙзҡ„е»әи®®пјҢжҲ‘д»¬еҸҜд»Ҙж №жҚ®е®ғзҡ„жҸҗзӨәиҝӣиЎҢдјҳеҢ–е·ҘдҪңгҖӮиҝҷйҮҢжҲ‘зҡ„зҪ‘з«ҷе·Із»ҸдјҳеҢ–еҲ°жңҖеҘҪжҢҮж ҮдәҶ(вҖўМҒвҢ„вҖўМҒа№‘)а«ӯвң§пјҢPage Speed ScoreиЎЁзӨәдҪ зҡ„жҖ§иғҪжөӢиҜ•еҫ—еҲҶпјҢ100/100иЎЁзӨәе·Із»ҸжІЎжңүйңҖиҰҒдјҳеҢ–зҡ„ең°ж–№гҖӮ

    дјҳеҢ–е®ҢжҜ•еҗҺеҶҚдҪҝз”Ёchorme devtoolsзҡ„networkзүҲеқ—жөӢйҮҸдёҖдёӢжҲ‘д»¬зҪ‘йЎөзҡ„зҷҪеұҸж—¶й—ҙиҝҳжңүйҰ–еұҸж—¶й—ҙпјҢжҳҜдёҚжҳҜеҫ—еҲ°дәҶеҫҲеӨ§зҡ„жҸҗеҚҮпјҹ

    д№ҹе°ұжҳҜиҜҙпјҢеңЁе’Ңиҝҗз»ҙеҜ№жҺҘзҡ„ж—¶еҖҷпјҢжҲ‘д»¬еҸӘйңҖиҰҒе°ҶдёҠйқўиҝҷеҮ иЎҢд»Јз Ғж”№дёәжҲ‘д»¬й…ҚзҪ®еҘҪзҡ„ж–ҮжЎЈеҸ‘йҖҒз»ҷд»–е°ұиЎҢдәҶпјҢе…¶д»–зҡ„дәӢжғ…пјҢиҝҗз»ҙе°Ҹе“ҘдјҡжҳҺзҷҪзҡ„пјҢдёҚз”ЁеӨҡиҜҙпјҢйғҪеңЁй…’йҮҢгҖӮ

    дҪҶжҳҜпјҢиҝҷеҮ иЎҢд»Јз ҒиҜҘжҖҺд№ҲеҺ»ж”№е‘ўпјҹйҰ–е…ҲжҲ‘д»¬еҫ—зҹҘйҒ“пјҢеңЁnginxдёӯпјҢжЁЎеқ—иў«еҲҶдёәдёүеӨ§зұ»пјҡhandlerгҖҒfilterе’ҢupstreamгҖӮиҖҢе…¶дёӯзҡ„upstreamжЁЎеқ—пјҢиҙҹиҙЈе®ҢжҲҗе®ҢжҲҗзҪ‘з»ңж•°жҚ®зҡ„жҺҘж”¶гҖҒеӨ„зҗҶе’ҢиҪ¬еҸ‘пјҢд№ҹжҳҜжҲ‘д»¬йңҖиҰҒеңЁеҸҚеҗ‘д»ЈзҗҶдёӯз”ЁеҲ°зҡ„жЁЎеқ—гҖӮжҺҘдёӢжқҘжҲ‘д»¬е°Ҷд»Ӣз»Қй…ҚзҪ®д»Јз ҒйҮҢзҡ„еҶ…е®№жүҖиЎЁзӨәзҡ„еҗ«д№ү

    4.3.1.upstreamй…ҚзҪ®дҝЎжҒҜпјҡ¶

    upstreamе…ій”®еӯ—еҗҺзҙ§и·ҹзҡ„ж ҮиҜҶз¬ҰжҳҜжҲ‘д»¬иҮӘе®ҡд№үзҡ„йЎ№зӣ®еҗҚз§°пјҢйҖҡиҝҮдёҖеҜ№иҠұжӢ¬еҸ·еңЁе…¶дёӯеўһж·»жҲ‘д»¬зҡ„й…ҚзҪ®дҝЎжҒҜгҖӮ

    ip_hash е…ій”®еӯ—пјҡжҺ§еҲ¶з”ЁжҲ·еҶҚж¬Ўи®ҝй—®ж—¶жҳҜеҗҰиҝһжҺҘеҲ°еүҚдёҖж¬ЎиҝһжҺҘзҡ„жңҚеҠЎеҷЁ

    serverе…ій”®еӯ—пјҡжҲ‘д»¬зңҹе®һжңҚеҠЎеҷЁзҡ„ең°еқҖпјҢиҝҷйҮҢзҡ„еҶ…е®№иӮҜе®ҡжҳҜйңҖиҰҒжҲ‘д»¬еҺ»еЎ«еҶҷзҡ„пјҢдёҚз„¶иҝҗз»ҙжҖҺд№ҲзҹҘйҒ“дҪ жҠҠйЎ№зӣ®ж”ҫеңЁйӮЈдёӘжңҚеҠЎеҷЁдёҠдәҶпјҢд№ҹдёҚзҹҘйҒ“дҪ е°ҒиЈ…дәҶдёҖеұӮnodeиҖҢеҫ—еҺ»зӣ‘еҗ¬3000з«ҜеҸЈгҖӮ

    4.3.2.serverй…ҚзҪ®дҝЎжҒҜ¶

    serverжҳҜnginxзҡ„еҹәжң¬й…ҚзҪ®пјҢжҲ‘д»¬йңҖиҰҒйҖҡиҝҮserverе°ҶжҲ‘д»¬е®ҡд№үзҡ„upstreamеә”з”ЁеҲ°жңҚеҠЎеҷЁдёҠгҖӮ

    listenе…ій”®еӯ—пјҡжңҚеҠЎеҷЁзӣ‘еҗ¬зҡ„з«ҜеҸЈ

    locationе…ій”®еӯ—пјҡе’ҢжҲ‘д»¬д№ӢеүҚеңЁnodeеұӮиҜҙеҲ°зҡ„и·Ҝз”ұжҳҜиө·еҗҢж ·зҡ„еҠҹиғҪпјҢиҝҷйҮҢжҳҜжҠҠз”ЁжҲ·зҡ„иҜ·жұӮеҲҶй…ҚеҲ°еҜ№еә”зҡ„upstreamдёҠ

    5.жӢ“еұ•йҳ…иҜ»¶

    зҪ‘з«ҷзҡ„жҖ§иғҪдёҺзӣ‘жөӢжҳҜдёҖйЎ№еӨҚжқӮзҡ„е·ҘдҪңпјҢиҝҳжңүеҫҲеӨҡеҫҲеӨҡеҗҺз»ӯзҡ„е·ҘдҪңпјҢжҲ‘д№ӢеүҚжүҖжҸҗеҲ°зҡ„иҝҷдәӣпјҢд№ҹеҸӘиғҪз®—жҳҜеҶ°еұұдёҖи§’пјҢеңЁзҶҹжӮүејҖеҸ‘и§„иҢғзҡ„еҗҢж—¶пјҢд№ҹйңҖиҰҒе®һи·өз»ҸйӘҢзҡ„з§ҜзҙҜгҖӮ

    еңЁзҝ»йҳ…дәҶи®ёеӨҡдёҺзҪ‘з«ҷжҖ§иғҪзӣёе…ізҡ„д№ҰзұҚеҗҺпјҢжҲ‘иҝҳжҳҜжӣҙй’ҹжғ…дәҺе”җж–ҮеүҚиҫҲзј–и‘—зҡ„гҖҠеӨ§еһӢзҪ‘з«ҷжҖ§иғҪзӣ‘жөӢгҖҒеҲҶжһҗдёҺдјҳеҢ–гҖӢпјҢйҮҢйқўзҡ„зҹҘиҜҶиҫғж–°пјҢеҲҮеҗҲе®һйҷ…пјҢиҮіе°‘жҲ‘иҜ»е®ҢдёҖйҒҚеҗҺеҫҲжңүж”¶иҺ·гҖҒйҶҚйҶҗзҒҢйЎ¶пјҢжҲ‘д№ҹеёҢжңӣеҜ№жҖ§иғҪж„ҹе…ҙи¶Јзҡ„иҜ»иҖ…еңЁзңӢе®ҢжҲ‘зҡ„ж–Үз« еҗҺиғҪеҺ»зҝ»зҝ»иҝҷжң¬и‘—дҪңгҖӮ

    иҝҷйҮҢз¬”иҖ…иҝҳе»әи®®еӨ§е®¶е№іж—¶жңүдәӢжІЎдәӢеҸҜд»ҘеӨҡеҺ»зңӢеҮ йҒҚйӣ…иҷҺеҶӣи§„пјҢиҷҪжҳҜиҖҒз”ҹеёёи°ҲпјҢдҪҶеҚҙеӯ—еӯ—зҸ зҺ‘гҖӮеҰӮжһңеӨ§е®¶иғҪзҶҹи®°дәҺеҝғжӣҙжҳҜеҶҚеҘҪдёҚиҝҮдәҶпјҢдј йҖҒй—Ёпјҡ
