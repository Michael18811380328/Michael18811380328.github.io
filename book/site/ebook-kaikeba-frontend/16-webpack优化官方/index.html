    webpackжҖ§иғҪдјҳеҢ– дјҳеҢ–ејҖеҸ‘дҪ“йӘҢ¶

    2021-10-29

    иҝҷйғЁеҲҶеҜ№еә”и§Ҷйў‘ж•ҷзЁӢдёӯ 456 дёүиҠӮпјҢеҺҹе§Ӣз¬”и®°жҜ”иҫғд№ұгҖӮ

    иҝҷдёүиҠӮи§Ҷйў‘иҫғй•ҝпјҢи®ІеёҲж°ҙе№ідёҖиҲ¬пјҢж №жҚ®е®һйҷ…жғ…еҶөеӯҰд№ гҖӮ

    дјҳеҢ–ејҖеҸ‘дҪ“йӘҢ¶

    жҸҗеҚҮж•ҲзҺҮпјӣдјҳеҢ–жһ„е»әйҖҹеәҰпјӣдјҳеҢ–дҪҝвҪӨдҪ“йӘҢ

    дјҳеҢ–иҫ“еҮәиҙЁйҮҸ¶

    дјҳеҢ–иҰҒеҸ‘еёғеҲ°зәҝдёҠзҡ„д»Јз ҒпјҢеҮҸе°‘вҪӨжҲ·иғҪж„ҹзҹҘеҲ°зҡ„еҠ иҪҪж—¶й—ҙпјӣжҸҗеҚҮд»Јз ҒжҖ§иғҪпјҢжҖ§иғҪеҘҪпјҢжү§вҫҸе°ұеҝ«

    зј©вј©е°ҸвҪӮж–Үд»¶иҢғеӣҙ

    Loader дјҳеҢ–loaderй…ҚзҪ®

    test include excludeдёүдёӘй…ҚзҪ®йЎ№жқҘзј©е°Ҹ loader зҡ„еӨ„зҗҶиҢғеӣҙ жҺЁиҚҗ include

    include: path.resolve(__dirname, "./src"),

    дјҳеҢ– resolve.modules й…ҚзҪ®¶

    resolve.modules вҪӨдәҺй…ҚзҪ® webpack еҺ»е“ӘдәӣвҪ¬еҪ•дёӢеҜ»жүҫз¬¬дёүвҪ…жЁЎеқ—пјҢй»ҳи®ӨжҳҜ['node_modules']

    еҜ»жүҫз¬¬дёүвҪ…жЁЎеқ—пјҢй»ҳи®ӨжҳҜеңЁеҪ“еүҚйЎ№вҪ¬зӣ®вҪ¬зӣ®еҪ•дёӢзҡ„node_modulesвҫҘйқўеҺ»жүҫпјҢеҰӮжһңжІЎжңүжүҫеҲ°пјҢе°ұдјҡеҺ»дёҠвјҖзә§зӣ®еҪ•../node_modulesжүҫпјҢеҶҚжІЎжңүдјҡеҺ»../../node_modulesдёӯжүҫпјҢд»ҘжӯӨзұ»жҺЁпјҢе’ҢNode.jsзҡ„жЁЎеқ—еҜ»жүҫжңәеҲ¶еҫҲзұ»дјјгҖӮ

    еҰӮжһңжҲ‘д»¬зҡ„з¬¬дёүвҪ…жЁЎеқ—йғҪе®үиЈ…еңЁдәҶпҰәйЎ№вҪ¬ж №зӣ®еҪ•дёӢпјҢе°ұеҸҜд»ҘзӣҙжҺҘжҢҮжҳҺиҝҷдёӘпӨ·еҫ„гҖӮ

    дјҳеҢ– resolve.alias й…ҚзҪ®¶

    resolve.aliasй…ҚзҪ®йҖҡиҝҮеҲ«еҗҚжқҘе°ҶеҺҹеҜје…ҘпӨ·еҫ„жҳ е°„жҲҗвјҖдёӘж–°зҡ„еҜјвјҠпӨ·еҫ„

    жӢҝreactдёәдҫӢпјҢжҲ‘д»¬еј•вјҠе…Ҙзҡ„reactеә“пјҢвјҖиҲ¬еӯҳеңЁдёӨеҘ—д»Јз Ғ

    cjs йҮҮвҪӨcommonJSи§„иҢғзҡ„жЁЎеқ—еҢ–д»Јз ҒпјҢumd е·Із»Ҹжү“еҢ…еҘҪзҡ„е®Ңж•ҙд»Јз ҒпјҢжІЎжңүйҮҮвҪӨжЁЎеқ—еҢ–пјҢеҸҜд»ҘзӣҙжҺҘжү§вҫҸ

    й»ҳи®Өжғ…еҶөдёӢпјҢwebpackдјҡд»ҺвјҠеҸЈвҪӮд»¶ ./node_modules/bin/react/index ејҖе§ӢйҖ’еҪ’и§Јжһҗе’ҢеӨ„зҗҶдҫқиө–зҡ„ж–Үд»¶гҖӮжҲ‘д»¬еҸҜд»ҘзӣҙжҺҘжҢҮе®ҡвҪӮд»¶пјҢйҒҝе…ҚиҝҷеӨ„зҡ„иҖ—ж—¶гҖӮ

    module.exports={
  resolve:{
    modules: [path.resolve(__dirname, "./node_modules")]
  }
}

    alias: {
  "@": path.join(__dirname, "./pages"),
  "react": path.resolve(__dirname, "./node_modules/react/umd/react.production.min.js"),
  "react-dom": path.resolve( __dirname, "./node_modules/react-dom/umd/react-dom.production.min.js")
}

    дјҳеҢ– resolve.extensions¶

    й…ҚзҪ® resolve.extensions еңЁеҜје…ҘиҜӯеҸҘжІЎеёҰж–Үд»¶еҗҺзјҖж—¶пјҢwebpack дјҡвҫғиҮӘеҠЁеёҰдёҠеҗҺзјҖеҗҺпјҢеҺ»е°қиҜ•жҹҘжүҫж–Үд»¶жҳҜеҗҰеӯҳеңЁгҖӮ

    й»ҳи®ӨеҖј:

    extensions:['.js','.json','.jsx','.ts']

    еҗҺзјҖе°қиҜ•пҰңиЎЁе°ҪпҘҫзҡ„е°Ҹ

    еҜје…ҘиҜӯеҸҘе°ҪйҮҸпҘҫзҡ„еёҰдёҠеҗҺзјҖгҖӮ

    дҪҝвҪӨ externals дјҳеҢ– cdn йқҷжҖҒиө„жәҗпјҡе…¬еҸёжңүcdn:йқҷжҖҒиө„жәҗжңүйғЁзҪІеҲ°cdn жңүй“ҫжҺҘдәҶпҰә

    жҲ‘зҡ„bundleвҪӮд»¶вҫҘпјҢе°ұдёҚвҪӨжү“еҢ…иҝӣеҺ»иҝҷдёӘдҫқиө–пҰәпјҢдҪ“з§Ҝдјҡе°Ҹ

    жҲ‘д»¬еҸҜд»Ҙе°ҶвјҖдәӣJSвҪӮж–Үд»¶еӯҳеӮЁеңЁ CDN дёҠ(еҮҸе°‘ Webpack жү“еҢ…еҮәжқҘзҡ„ js дҪ“з§Ҝ)пјҢеңЁ index.html дёӯйҖҡиҝҮж Үзӯҫеј•е…ҘпјҢеҰӮ:
