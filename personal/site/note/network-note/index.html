    и§ЈеҶіиҝҮзЁӢпјҡе…ҲжҹҘзңӢ dtable-web е’Ң dtable-server зҡ„ж—Ҙеҝ—пјҲdtable-server дёӯжҳҫзӨәзј–иҜ‘й”ҷиҜҜпјүжң¬иҙЁдёҠпјҡеӣ дёәеңЁ docker еӨ–йғЁзҺҜеўғжү§иЎҢ npm installпјҢdtable-server жҹҗдәӣз¬¬дёүж–№дҫқиө–еә“дҪҝз”ЁCиҜӯиЁҖзј–иҜ‘пјҢжІЎжңүзј–иҜ‘еҲ° docker еҶ…йғЁпјҲеҚідҪҝе®үиЈ…е…¶д»–з¬¬дёүж–№еә“пјҢд№ҹдјҡеҪұе“Қе·Іжңүзҡ„иҝҷдёӘзү№ж®Ҡзҡ„еә“пјүгҖӮжүҖд»ҘйҖ жҲҗ server ж— жі•зј–иҜ‘пјҢжңҚеҠЎдёҚжӯЈеёёгҖӮnginx еҸҚеҗ‘д»ЈзҗҶжңҚеҠЎеҷЁе·Із»ҸеӨ„зҗҶдәҶи·ЁеҹҹпјҢдҪҶжҳҜе·ІжңүжңҚеҠЎжІЎиө·жқҘпјҢжүҖд»Ҙз•ҢйқўжҳҫзӨәзҡ„жҳҜи·ЁеҹҹпјҲжүҫдёҚеҲ°еҜ№еә”зҡ„жңҚеҠЎпјүгҖӮ

    жңҖеҗҺи§ЈеҶіпјҡеңЁ docker еҶ…йғЁеҲ йҷӨ node_modules з„¶еҗҺйҮҚж–° npm install ејҖеҗҜжңҚеҠЎпјҢжӯЈеёёдҪҝз”ЁгҖӮ

    жҖ»з»“пјҡз•Ңйқўзҡ„жҠҘй”ҷдёҚдёҖе®ҡжҳҜзңҹе®һзҡ„еҺҹеӣ пјҢйңҖиҰҒжҹҘзңӢж—Ҙеҝ—гҖӮnginx йңҖиҰҒеӨҡдәҶи§ЈгҖӮ

    еӯҰдјҡзҡ„¶

    SMTP е’Ң POP3 е’Ң IMAP зҡ„еҢәеҲ«¶

    зӣёеҗҢзӮ№пјҡиҝҷдёүдёӘйғҪжҳҜиҝҗиЎҢеңЁйӮ®д»¶жңҚеҠЎеҷЁзҡ„жңҚеҠЎпјҲдҫӢеҰӮQQйӮ®д»¶жңҚеҠЎеҷЁпјҢ163йӮ®д»¶жңҚеҠЎеҷЁзӯүпјү

    SMTP жңҚеҠЎпјҡеӨ„зҗҶеҸ‘йҖҒйӮ®д»¶гҖӮеҰӮжһңеҸ‘д»¶дәәе’Ңж”¶д»¶дәәжҳҜеҗҢдёҖдёӘеҹҹпјҲйғҪжҳҜ163иҮӘе·ұзҡ„еҹҹпјүпјҢйӮЈд№ҲзӣҙжҺҘиҪ¬пјӣеҰӮжһңдёҚжҳҜеҗҢдёҖдёӘеҹҹпјҲ163еҸ‘з»ҷQQпјүпјҢйҰ–е…ҲиҰҒиҝӣиЎҢ DNS жҹҘиҜўпјҢз„¶еҗҺжүҫеҲ° host е№¶еҸ‘йҖҒгҖӮ

    POP3 е’Ң IMAP пјҡеӨ„зҗҶжҺҘеҸ—йӮ®д»¶гҖӮPOP3 жҜ”иҫғж—§пјҢIMAP иҫғж–°гҖӮ

    POP3пјҡжң¬ең°иҜ»еҲ°йӮ®д»¶еҗҺпјҢйӮ®д»¶д»ҺжңҚеҠЎеҷЁдёҠеҲ йҷӨпјҢйӮ®д»¶иҝӣиЎҢжң¬ең°зј–иҫ‘гҖӮеҰӮжһңеӨҡдёӘз»Ҳз«Ҝй“ҫжҺҘжңҚеҠЎеҷЁжңүй—®йўҳгҖӮ

    IMAPпјҡжң¬ең°иҜ»еҲ°йӮ®д»¶еҗҺпјҢйӮ®д»¶дёҚдјҡд»ҺжңҚеҠЎеҷЁдёҠеҲ йҷӨгҖӮ

    иҜҰжғ…еҸӮиҖғпјҡhttps://www.zhihu.com/question/24605584

    иҝҷдёӘе’ҢеүҚз«Ҝе…іиҒ”дёҚеӨ§пјҢдәҶи§ЈеҚіеҸҜ

    RestAPI дёҺ GraphQL¶

    з®ҖеҚ•е’ҢеҗҢдәӢеҲҶдә«дәҶдёӨз§ҚзҪ‘з»ңжҠҖжңҜеҚҸи®®пјҢREST API е’Ң GraphQLгҖӮ

    restAPI жҳҜж—©жңҹзҡ„еҚҸи®®гҖӮеңЁзҪ‘з»ңиҜ·жұӮдёӯпјҢurl дёӯиЎЁзӨәиҜ·жұӮзҡ„иө„жәҗеҗҚз§°пјҢиҜ·жұӮзҡ„ж–№жі•еҜ№еә”ж•°жҚ®еә“зҡ„еўһеҲ ж”№жҹҘж“ҚдҪңпјҲеӣһжғідёҖдёӢеёёи§Ғзҡ„еӣӣз§ҚиҜ·жұӮж–№жі•пјҢget post delete put зҡ„зү№зӮ№пјҢд»ҘеҸҠ head option иҜ·жұӮж–№жі•пјүгҖӮиҝҷж ·зҡ„еҘҪеӨ„жҳҜ URL зӣҙжҺҘе…іжіЁиө„жәҗпјҢдёҚйңҖиҰҒе…іжіЁжҖҺд№Ҳж“ҚдҪңиө„жәҗпјҢзӣ®еүҚйЎ№зӣ®еӨ§йғЁеҲҶдҪҝз”ЁиҝҷдёӘеҚҸи®®гҖӮ

    GraphQLжҳҜж–°дёҖд»Јзҡ„еҚҸи®®гҖӮжҜҸдёҖдёӘиҜ·жұӮйғҪзұ»дјј POST иҜ·жұӮпјҢеүҚз«ҜжҠҠйңҖиҰҒзҡ„еҸӮж•°йҖҡиҝҮ POST зұ»дјј JSON еҸ‘з»ҷеҗҺз«ҜпјҢз„¶еҗҺеҗҺз«Ҝж №жҚ®йңҖиҰҒзҡ„еҸӮж•°иҝ”еӣһз»ҷеүҚз«ҜпјҲеҗҺз«ҜйңҖиҰҒжҹҘиҜўпјүгҖӮеҘҪеӨ„пјҡURLеӣәе®ҡзҡ„пјҢдёҚдјҡйў‘з№ҒеҸҳеҢ–гҖӮеҫҲеӨҡиө„жәҗеҸҜд»ҘеңЁдёҖдёӘ Query дёӯиҺ·еҸ–пјҢиҝҷж ·жңҚеҠЎеҷЁе№¶иЎҢиҜ·жұӮеҺӢеҠӣиҫғе°ҸгҖӮжңҚеҠЎеҷЁж— йңҖиҝ”еӣһдёҚйңҖиҰҒзҡ„еӯ—ж®өпјҲеүҚз«ҜйңҖиҰҒз”ЁжҲ·е§“еҗҚпјҢеҗҺз«ҜдјҡжҹҘиҜўдёҖж¬Ўж•°жҚ®еә“пјҢжҠҠз”ЁжҲ·еҫҲеӨҡдҝЎжҒҜжӢҝеҲ°е№¶иҝ”еӣһпјүгҖӮиҝҷдёӘд№ҹжңүдёҚи¶іпјҢйңҖиҰҒи®ҫзҪ®еҫҲеӨҡзҡ„жҹҘиҜўе’Ңй”ҷиҜҜеӨ„зҗҶпјҲеүҚз«ҜжҹҘиҜўдёӨдёӘеҸӮж•°пјҢдёҖдёӘеҮәй”ҷпјҢжҖҺд№Ҳиҝ”еӣһпјӣж•°жҚ®йҮҸиҫғеӨ§пјҢдј иҫ“жӣҙеҠ ж¶ҲиҖ—ж—¶й—ҙвҖ”вҖ”зҪ‘з»ңиүҜеҘҪзҡ„жғ…еҶөдёӢдёҚйңҖиҰҒиҖғиҷ‘иҝҷдёӘпјүгҖӮ

    еҢәеҲ«пјҡvps дә‘жңҚеҠЎеҷЁ зү©зҗҶжңҚеҠЎеҷЁ¶

    зү©зҗҶжңҚеҠЎеҷЁпјҡзңҹе®һзҡ„жңҚеҠЎеҷЁпјҢз”ЁдәҺеӨ§дјҒдёҡпјҢеҸҜйқ жҖ§жҜ”иҫғй«ҳпјҢйңҖиҰҒдё“дәәз»ҙжҠӨгҖӮеҶ…йғЁз»ҙжҠӨжҜ”иҫғе®үе…ЁзЁіе®ҡпјҲдҫӢеҰӮйҮ‘иһҚз”өеҠӣйҖҡдҝЎиЎҢдёҡпјүгҖӮ

    vps: Virtual Private ServerпјҲиҷҡжӢҹз§ҒжңүжңҚеҠЎеҷЁпјүпјҢдёҖиҲ¬жһ¶и®ҫеңЁдә‘з«ҜпјҢжңүеӣәе®ҡзҡ„еёҰе®ҪпјҢдёӘдәәеҚҡе®ўеҸҜд»Ҙжҗӯе»әеңЁиҝҷйҮҢгҖӮдјҒдёҡзҡ„зҪ‘з«ҷдёҚйҖӮе®ңжҗӯе»әгҖӮ

    дә‘жңҚеҠЎеҷЁпјҡжҳҜ vps зҡ„еҠ ејәзүҲпјҢеёҰе®ҪеҸҜеҸҳпјҢеӣҪеҶ…дё»иҰҒжҳҜеҚҺдёәдә‘пјҢйҳҝйҮҢдә‘пјҢеӣҪеӨ–жҳҜдәҡй©¬йҖҠгҖӮ
