下午大题¶

下午 150 分钟，5道大题

1、数据流图

2、数据库设计

3、UML 建模

4、C 语言(算法)

5、Java 语言（面向对象）

通常C语言算法比较难，不容易得分。其他题目每一个15分，要求拿到 12分左右。

数据流图¶

这个对应下午第一道大题（数据结构）

结构化分析-补充数据流

​

​

数据流图题目考点和做题思路：阅读理解

1、实体（软件外部的人或者设别，用方形表示），数据库（数据最终存放到哪里）

2、顶层数据流，0层数据流：顶层就是概括，0层就是细节图

3、先看一下四个小题，然后看一下0层数据流和顶层数据流图，大概理解内容

4、然后看题干给的细节信息，补充对应的实体或者数据库模块（前两个小题）

5、后两个小题：找到缺失的数据流或者错误的数据流，这个需要一个一个看细节。

UML建模¶

UML 建模，就是面向对象设计的建模，实际上有多个类型的图，主要是：用例图+类图

元素包括：事物（类）、关系（接口）、图

把握主要点
